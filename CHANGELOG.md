# Changelog

All notable changes to Quaternuli are documented here.

## [0.1.0] — 2026-04-03

### Added
- Seed notebook with three phases: Capture → Tend → Harvest
- The Gardener — AI assistant powered by Claude (Anthropic), streaming via SSE
- Code editor with CodeMirror 6, syntax highlighting for 8 languages
- Code Gardener — AI code review for the code editor
- Local-first storage via IndexedDB
- Export seeds as Markdown, plain text, styled HTML, or JSON
- Responsive layout — mobile, tablet, and desktop
- Swiss International Style design system
- MIT license
